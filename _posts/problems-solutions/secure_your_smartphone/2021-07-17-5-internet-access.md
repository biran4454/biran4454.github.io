### Internet Access
**Problem:** Many browsers and search engines - such as Google Chrome or Microsoft Edge with Bing - track your search history and have little protection against tracking. Some have the option to send a DNT (Do Not Track) request, which asks websites not to track you, however most websites ignore that request and continue to track your browsing habits and history.<br />
**Solution:** Use a browser such as Firefox, and ensure that you have the security settings to the "strict" setting. If you use Firefox don't forget to set the usage data collection to off. Make sure to use a search engine such as Duckduckgo. Set https-only mode to be always on. Install addons such as Privacy Badger, uBlock Origin, decentralise, Privacy Possum, and fingerprint blockers.<br />
